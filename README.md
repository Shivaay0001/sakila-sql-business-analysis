# 🎬 Sakila SQL Business Analysis

## 📊 Project Overview

This project analyzes the **Sakila movie rental database** using SQL to extract meaningful business insights related to customer behavior, revenue trends, and inventory performance.

## 🎯 Objectives

* Analyze customer behavior and rental trends
* Identify top customers and popular movies
* Evaluate revenue by store and category
* Study monthly rental and revenue trends
* Generate data-driven business insights


## 🗂️ Dataset Description

The Sakila database represents a real-world movie rental system containing:

* Customer information
* Rental transactions
* Payment data
* Film details
* Inventory and store data


## 🛠️ Tools Used

* MySQL Workbench
* SQL


## 📈 Key Metrics

* 👥 Total Customers: **599**
* 🎟️ Total Rentals: **16,044**
* 💰 Total Revenue: **$67,407**


## 🔍 Key Analysis Performed

* Total customers, rentals, and revenue
* Top customers by spending
* Most rented movies
* Revenue by category
* Monthly rental trends
* Store performance comparison
